# Flask Book App (Unfinished)

# Prerequisites
Before running the application, make sure you have the following installed on your machine:
- [Python 3](https://www.python.org/)

# Using Tech
- Flask
- Flask-SQLAlchemy
- Jinja2


## How to Install
```bash
pip install -r requirements.txt
```

## How to Run
```bash
python app.py
```
